# ECC: The Agent Harness Performance Optimization System

> Sources: affaan-m, Unknown date
> Raw: [ECC](../../raw/ai-agent-harnesses/ecc.md)

## Overview

Positioned as an "operating system" for AI agent harnesses that works across Claude Code, Cursor, OpenCode, Codex, Gemini, Zed, and GitHub Copilot, bundling skills, "instincts," memory optimization, continuous learning, and security scanning (AgentShield) into one system.

## Features

66 agents across 12+ language ecosystems, 268+ skills, 84 legacy command shims for backward compatibility, session management with persistence. Three installation paths (plugin, manual installer, minimal profile) that the README says must not be mixed, since stacking creates duplicate skills and broken configs. v2.0.0 adds a control-pane substrate, session adapters, MCP inventory management, and orchestrator tools. MIT licensed with an optional commercial "ECC Pro" GitHub App for private repos.

## Note on Claims

The README reports over 211,000 stars and 32,500 forks, figures that would place this niche multi-harness project above nearly every repository on GitHub. This could not be verified independently at time of collection and should be treated with skepticism pending confirmation.
