# Awesome Harness Engineering

> Sources: ai-boost, Unknown date
> Raw: [Awesome Harness Engineering](../../raw/ai-agent-harnesses/awesome-harness-engineering.md)

## Overview

A curated resource list treating "harness engineering", the scaffolding around an AI agent (context delivery, tool interfaces, planning artifacts, verification loops, memory, sandboxes), as the primary lever for whether an agent succeeds on real tasks, more so than the underlying model.

## Structure and Content

The list is organized around a full harness design taxonomy: Foundations (essays from OpenAI, Anthropic, Google, IBM, Martin Fowler, and academia), Design Primitives split by concern (agent loop, planning/decomposition, context delivery and compaction, tool design, skills and MCP, permissions/authorization, memory and state, orchestration, verification/CI, observability, debugging, human-in-the-loop), Reference Implementations, Security/Sandboxing, Evals, Templates, and Production Infrastructure.

Featured resources span vendor guidance (OpenAI's Codex harness docs, Anthropic's context engineering and tool design guides, Google's Agent Development Kit), Microsoft tooling (TaskWeaver, LLMLingua), academic planning/multi-agent papers, and frameworks like LangChain/LangGraph. Licensed CC0.
