# Multica

> Sources: multica-ai, Unknown date
> Raw: [Multica](../../raw/ai-agent-harnesses/multica.md)

## Overview

An "open-source managed agents platform" that treats AI coding agents as team members with assignments, comments, and accumulating skills, rather than one-off prompt sessions. Tagline: "Your next 10 hires won't be human."

## Features

Agents as Teammates (assignments, comments, issue creation, proactive blocker reporting); Squads (agent+human groups led by a leader agent); Autonomous Execution with WebSocket progress streaming; Autopilots (cron/webhook-triggered recurring work); Reusable Skills that compound across the team; a Unified Runtimes dashboard for local daemons and cloud runtimes; Multi-Workspace isolation.

Compatible with 14 agent runtimes including Claude Code, Codex, Cursor Agent, and Antigravity. Stack: Next.js 16 frontend, Go/Chi backend with sqlc, PostgreSQL 17 + pgvector. Installable via Homebrew, curl, PowerShell, or self-hosted Docker.
