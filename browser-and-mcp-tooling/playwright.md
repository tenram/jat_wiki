# Playwright

> Sources: Microsoft, Unknown date
> Raw: [Playwright](../../raw/browser-and-mcp-tooling/playwright.md)

## Overview

A cross-browser (Chromium, Firefox, WebKit) automation library from Microsoft, used for testing, scripting, and increasingly as the execution layer for AI browsing agents. Supports TypeScript, Python, .NET, and Java.

## Features

Auto-waiting with retryable assertions (removes flaky-test timeouts), per-test isolated browser contexts, resilient role/label/testid-based locators, parallel/sharded execution, a Trace Viewer, a VS Code extension, and accessibility snapshots specifically designed for AI agent consumption. Ships an MCP server integration (see [Playwright MCP](playwright-mcp.md)) and a CLI aimed at coding agents. Claimed adoption: 92k+ GitHub stars; used by VS Code, Bing, Outlook, and others.
