# Claude Design

> Sources: Anthropic, Unknown date
> Raw: [Get Started with Claude Design](../../raw/claude-code-ecosystem/claude-design.md)

## Overview

A conversational design surface (beta, Pro/Max/Team/Enterprise plans) with a chat-left, live-canvas-right layout, generating designs, prototypes, and presentations that inherit an organization's design system (colors, typography, components) automatically.

## Workflow

Create a project (auto-inherits org design system) → import design system from GitHub/design files/codebase → add context (screenshots, assets) → describe desired output in chat → review generated canvas → iterate via chat, inline comments, or direct canvas edits → export (PDF, PPTX, HTML) or hand off to Claude Code via `/design-sync`. Includes built-in accessibility review. Usage counts against standard Claude usage limits.
