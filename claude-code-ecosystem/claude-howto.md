# Claude How To

> Sources: Luong NGUYEN, Unknown date
> Raw: [Claude How To](../../raw/claude-code-ecosystem/claude-howto.md)

## Overview

A visual, example-driven Claude Code guide positioned as filling a gap in official docs, which the author argues describe features in isolation without showing combined workflows or a guided learning sequence.

## Content

Ten modules: Slash Commands, Memory, Skills, Subagents, MCP Protocol, Hooks (29 event types documented), Plugins, Checkpoints, Advanced Features (planning modes, background tasks), and CLI Reference. Full path claimed at 11-13 hours; a single slash-command template gives value in 15 minutes. Includes a self-assessment tool for personalizing the learning path. MIT licensed.

Author (Luong NGUYEN, Paris-based backend/AI engineer) maintains related tooling: `asm` (universal skill manager for AI coding agents), a generic `skills` framework, `context-stats` (Claude Code usage analysis), `sleek-ui` (design system framework), and `ccl` (model-agnostic model switching).
