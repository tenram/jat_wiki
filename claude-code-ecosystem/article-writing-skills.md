## Overview

A small skill collection providing LLM system prompts that capture named engineers'/researchers' explanatory *thinking patterns* (currently just an "Andrej Karpathy" skill: first-principles, code-forward explanations of ML systems), not their writing style.

## Design and Ethics

Compatible with Claude Code (`~/.claude/skills/`), Claude.ai, ChatGPT, or any LLM accepting markdown system prompts. The project explicitly distinguishes thinking-framework emulation from style mimicry: "the goal is not to produce prose that pretends to be authored by them," and cautions against ghostwriting, impersonation, or undisclosed AI-assisted publishing. MIT licensed; single skill at time of collection, contributions welcomed.
